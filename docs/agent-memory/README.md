# Agent Memory

This folder is the shared project memory for day-by-day progress tracking.

Rules:
- Start from `docs/agent-memory/LATEST.md`.
- Create one file per day in `docs/agent-memory/daily/`.
- Use ISO dates for filenames: `YYYY-MM-DD.md`.
- Record decisions, deployed changes, open issues, and follow-up items.
- Keep entries short and factual.
- Before starting a new task, read the latest daily file.
- After important work, append a short update.

Suggested sections:
- Context
- Decisions
- Changes
- Deployments
- Open items

Current convention:
- Latest pointer lives in `docs/agent-memory/LATEST.md`
- Daily notes live in `docs/agent-memory/daily/`
- One file per day
- New daily files should be created from `docs/agent-memory/templates/daily-template.md`
