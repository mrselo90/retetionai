# 2026-04-08

## Context
- Main focus: Refine Shopify embedded app product setup UX on `/app/products`.
- Environment: local workspace (`retention-agent-ai`), Shopify embedded app production route.

## Decisions
- Keep `/app/products` as a single workflow, but move the selected product setup area below the full product list instead of using a side-by-side split layout.
- Refine the products list into richer inline setup rows so the selected product expands directly under its own row instead of opening in a detached block.
- Proposed multilingual direction:
  - remove merchant-facing `default source language` selection
  - detect source language automatically from collected product content
  - let merchants choose only customer service / response languages
  - treat multilingual enrichment as a required backfill job whenever new service languages are added
